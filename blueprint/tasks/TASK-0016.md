# TASK-0016

## Title

Implement Unit of Work Contract

---

## Goal

Introduce the Unit of Work abstraction for the SafetyMAIN Core.

The Unit of Work coordinates repositories and guarantees atomic execution of application use cases.

No persistence implementation is required.

---

## Create

backend/core/contracts/unit_of_work.py

backend/core/domain/repositories/__init__.py

---

## Requirements

Create a UnitOfWork protocol using typing.Protocol.

The contract shall expose:

- commit()
- rollback()

It shall provide access to repositories through typed properties.

No implementation.

No SQLAlchemy.

No PostgreSQL.

No transactions.

No business logic.

---

## Acceptance Criteria

✓ Protocol created

✓ Storage agnostic

✓ Compatible with Clean Architecture

✓ No implementation

✓ Imports validated
