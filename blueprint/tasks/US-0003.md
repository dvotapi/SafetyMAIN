# US-0003

## Title

Archive and Restore Knowledge Object

---

## Goal

Implement lifecycle management for Knowledge Objects.

The implementation shall support:

- Archive
- Restore

while preserving immutable version history.

---

## Requirements

Implement:

ArchiveKnowledgeObjectCommand

RestoreKnowledgeObjectCommand

ArchiveKnowledgeObjectHandler

RestoreKnowledgeObjectHandler

KnowledgeObjectService

---

## Rules

Archive changes status to ARCHIVED.

Restore changes status to ACTIVE.

Each operation creates a new version.

History is preserved.

---

## Repository

Support:

archive()

restore()

---

## Tests

Verify:

Create

↓

Archive

↓

Status == ARCHIVED

↓

Restore

↓

Status == ACTIVE

↓

History contains every previous version

---

## Acceptance Criteria

✓ Archive works

✓ Restore works

✓ Version history preserved

✓ Tests pass
