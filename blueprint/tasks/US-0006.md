# US-0006

## Title

Traverse Knowledge Object Relations

## Goal

Add read capabilities for navigating directed Knowledge Object relations.

The system shall support retrieving:

- outgoing relations;
- incoming relations;
- directly connected Knowledge Objects.

This task implements one-hop traversal only.

Recursive graph traversal is out of scope.

## Acceptance Criteria

- Outgoing traversal implemented.
- Incoming traversal implemented.
- One-hop connected-object traversal implemented.
- Direction filtering implemented.
- Relation type filtering implemented.
- Organization isolation preserved.
- Deleted objects are handled according to the defined rules.
- Duplicate neighbours are removed.
- Results are deterministic and immutable.
- Read handlers do not commit.
- Repository and architecture boundaries remain clean.
- All tests pass.
- Ruff passes.
- Import validation passes.
- Architecture tests pass.
