# US-0001

## Title

Create Knowledge Object

---

## Goal

Implement the first complete use case of the SafetyMAIN platform.

The use case shall demonstrate the interaction between:

- Domain
- Application
- Unit of Work
- Repository
- InMemory Adapter

without introducing external infrastructure.

---

## Scope

Implement:

Application Command

Application Handler

InMemory Repository

InMemory UnitOfWork

Use existing Knowledge Object model.

---

## Requirements

The handler shall:

- receive a CreateKnowledgeObject command
- create a Knowledge Object
- store it in memory
- commit UnitOfWork
- return the created object

---

## Constraints

No FastAPI

No PostgreSQL

No SQLAlchemy

No Redis

No AI

No Metadata Engine

No Rules

No Events

---

## Tests

Implement an integration test proving that:

Create Command

↓

Handler

↓

UnitOfWork

↓

Repository

↓

Knowledge Object

works successfully.

---

## Acceptance Criteria

✓ End-to-end use case works

✓ Repository abstraction respected

✓ UnitOfWork used

✓ Tests pass

✓ Infrastructure independent
