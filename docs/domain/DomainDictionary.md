# TASK-0006

## Title

Create SafetyMAIN Domain Dictionary

---

## Priority

Critical

---

## Epic

Architecture Foundation

---

## Goal

Create the official domain vocabulary of the SafetyMAIN platform.

This document becomes the single source of truth for all business terminology used throughout the project.

Every future architecture document, source code file, API, database schema, AI prompt and user interface must use the terminology defined here.

---

## Context

SafetyMAIN is a long-term enterprise platform.

Different terminology used by different developers will inevitably create inconsistencies.

The platform therefore adopts a formal Domain Dictionary.
